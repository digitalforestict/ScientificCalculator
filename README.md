# ScientificCalculator
comprehensive scientific calculator with advanced mathematical functions, all in a single HTML
Key Features of This Advanced Scientific Calculator:
1. Three Calculator Modes
Standard Mode: Basic arithmetic operations

Scientific Mode: Advanced mathematical functions (trigonometry, logarithms, exponents)

Programming Mode: Bitwise operations and number system conversions

2. Comprehensive Mathematical Functions
Basic Operations: +, −, ×, ÷, %, ±, 1/x, x², x³, √, ∛

Trigonometric Functions: sin, cos, tan, sinh, cosh, tanh (with inverses)

Logarithmic Functions: log, ln, log₂, eˣ, 10ˣ

Advanced Functions: factorial, absolute value, modulo, power (xʸ)

Constants: π (pi), e (Euler's number)

3. Programming Functions
Bitwise Operations: AND, OR, XOR, NOT, left shift, right shift

Number System Conversions: Decimal ↔ Binary, Decimal ↔ Hexadecimal, Decimal ↔ Octal

4. Memory System
Multiple memory slots (M1, M2, etc.)

Memory operations: MC (clear), MR (recall), M+ (add), M- (subtract)

Click memory items to recall values

Persistent storage using localStorage

5. Advanced Features
Calculation History: Stores previous calculations with timestamps

Dark/Light Theme: Toggle with persistent preference

Angle Mode: Radians/Degrees (toggle with Ctrl+D)

Expression Display: Shows full calculation expression

Tooltips: Hover over buttons for function descriptions

Keyboard Support: Full keyboard navigation

6. Professional UI/UX
Clean, modern design with purple color scheme

Responsive layout for all devices

Visual feedback with animations

Tooltips for all scientific functions

Expression preview for complex calculations

7. Technical Excellence
Local Storage: Saves memory, history, and theme preferences

Error Handling: Validates inputs for mathematical constraints

Precision Control: Limits decimal places appropriately

Performance: Efficient calculation algorithms

8. User Experience
Tabbed Interface: Easy switching between calculator modes

Clickable History: Reuse previous results by clicking

Live Expression Display: See your calculation as you build it

Visual Feedback: Animations for calculations and transitions

How to Use:
Basic Operations:
Enter numbers using the keypad

Select an operator (+, −, ×, ÷)

Enter second number

Press = to calculate

Scientific Functions:
Enter a number

Press the function button (sin, cos, log, etc.)

Result is calculated immediately

Memory Functions:
M+: Add current value to memory

MR: Recall last memory value

MC: Clear all memory

Click memory items in the panel to recall

Keyboard Shortcuts:
Numbers: 0-9

Operators: +, -, *, /

Equals: = or Enter

Clear: Escape or Delete

Delete: Backspace
Pi: Ctrl+P
Euler's number: Ctrl+E
Square root: Ctrl+S

Toggle Angle Mode: Ctrl+D

Switching Modes:
Use the tabs at the top to switch between:

Standard - Basic calculator

Scientific - Advanced math functions

Programming - Bitwise operations and conversions
